HTML Media

HTML5 introduced new media elements to handle multimedia content more effectively. The two primary media elements are <audio> and <video>.

1. The <audio> Element

The <audio> element is used to embed sound content in a web page. It supports multiple audio formats, including MP3, WAV, and OGG.

2. The <video> Element

The <video> element is used to embed video content in a web page. It supports multiple video formats, including MP4, WebM, and OGG.

Attributes of <audio> and <video> Elements

  • controls: Adds playback controls (play, pause, volume, etc.)
  • autoplay: Starts playing the media as soon as it is ready
  • loop: Repeats the media automatically when it ends
  • muted: Mutes the audio by default
  • poster (video only): Specifies an image to be shown while the video is downloading or until the user hits the play button

3. Embedding Other Media Types

For embedding other types of media, such as Flash or PDFs, the <embed> and <object> elements can be used. However, these are less common with the rise of HTML5.
