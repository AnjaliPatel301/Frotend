Table


HTML tables are used to organize and display data in a structured format using rows and columns. Tables are particularly useful for presenting tabular data, such as schedules, pricing information, or any other data that can be logically arranged in a grid.

  • Tables are created using the <table> tag in HTML.
  • The <tr> tag defines a table row, while the <td> tag defines a table cell (data).
  • The <th> tag is used to define a table header cell, which is typically bold and centered by default.
  • Tables can include additional elements such as <thead>, <tbody>, and <tfoot> to group header, body, and footer content respectively.
  • Tables can be styled using CSS to enhance their appearance, including borders, background colors, and spacing.
  • Proper use of tables improves data readability and organization on a webpage.
  • Tables should be used for tabular data only, not for layout purposes, to ensure accessibility and maintain semantic meaning.

Table Cells

Each table cell is defined by a td and a td tag.

Everything between td and td is the content of a table cell.
