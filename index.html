    What is html?

  • HTML stands for Hyper Text Markup Language
  • It is a markup language, not a programming language. This means it annotates text to define how it is structured and displayed by web browsers.
  • It organizes text, images, links, and media using tags and elements that browsers can interpret. As of 2025, over 95% of websites rely on HTML alongside CSS and JavaScript, making it a fundamental tool in modern web development.
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • In a nutshell, HTML is all about organizing and displaying information on a webpage. We can think of it as the bones or structure of a webpage.
  • HTML consists of a series of elements
  • "Hypertext" refers to links that connect web pages to one another, either within a single website or between websites. Links are a fundamental aspect of the Web. By uploading content to the Internet and linking it to pages created by other people, you become an active participant in the World Wide Web.
  • HTML elements tell the browser how to display the content
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

    HTML Basic Structure

  • <!DOCTYPE html>: Defines the document type and version of HTML
  • <html>: The root element that contains all other HTML elements
  • <head>: Contains meta-information about the document, such as the title and links to stylesheets
  • <title>: Sets the title of the webpage, which appears in the browser tab
  • <body>: Contains the content of the webpage, such as text, images, and links

    Web browser

  • A web browser is a software application used to access and view websites on the internet.
  • Examples of popular web browsers include Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, and Opera.
  • Web browsers interpret HTML, CSS, and JavaScript code to display web pages as intended by web developers.
  • Web browsers provide features such as bookmarks, history, tabs, and extensions to enhance the user experience.
  • Web browsers also have security features to protect users from malicious websites and online threats.

    HTML Elements

  • HTML elements are the building blocks of HTML documents.
  • They are used to structure and format content on a webpage.
  • HTML elements are represented by tags, which are enclosed in angle brackets (<>).
  • Most HTML elements have an opening tag and a closing tag, with the content in between.
  • Some HTML elements are self-closing and do not require a closing tag.
  • HTML elements can have attributes that provide additional information or modify their behavior.

    Common HTML Elements

  • <h1> to <h6>: Heading elements, with <h1> being the highest level and <h6> the lowest
  • <p>: Paragraph element, used to define a block of text
  • <a>: Anchor element, used to create hyperlinks to other webpages or resources
  • <img>: Image element, used to embed images in a webpage
  • <div>: Division element, used to group and style sections of content
  • <span>: Span element, used to apply styles to inline text
  • <ul>, <ol>, <li>: Unordered list, ordered list, and list item elements, used to create lists
  • <table>, <tr>, <td>: Table, table row, and table data elements, used to create tables

    HTML history

    HTML (HyperText Markup Language) has a rich history that dates back to the early days of the World Wide Web. Here is a brief overview of its evolution:

  • 1989: Tim Berners-Lee, a British scientist, invented the World Wide Web while working at CERN. He proposed a system for sharing and linking documents using hypertext.
  • 1991: The first version of HTML (HTML 1.0) was introduced. It was a simple markup language that allowed for basic text formatting and hyperlinks.
  • 1995: HTML 2.0 was released, which included new features such as tables, forms, and image maps.
  • 1997: HTML 3.2 was introduced, adding support for scripting languages like JavaScript and multimedia elements like audio and video.
  • 1999: HTML 4.01 was released, which became the standard for web development for many years. It introduced features like CSS (Cascading Style Sheets) for styling and improved support for accessibility.
  • 2008: XHTML 1.1 was introduced as a stricter version of HTML, combining XML syntax with HTML elements.
  • 2014: HTML5 was officially published as a W3C Recommendation. It brought significant improvements, including new semantic elements (like <article>, <section>, <nav>), multimedia support (like <video> and <audio>), and APIs for web applications.
  • Present: HTML continues to evolve with ongoing updates and new features being added to enhance web development capabilities.
