minlength attribute

The minlength attribute specifies the minimum number of characters allowed in an element .

This attribute can be used with the following input types: text, email, password, search, tel, url

Definition and Usage

The minlength attribute specifies the minimum number of characters allowed in an element.

Tip: Use the minlength attribute to ensure that users enter enough characters.

This attribute can be used with the following input types: text, email, password, search, tel, url
