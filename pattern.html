pattern attribute

The pattern attribute is used to specify a regular expression that the input field's value must match in order for the form to be submitted.

This attribute can be used with the following input types: text, search, url, tel, email, and password

If the user enters a value that does not match the specified pattern, the form will not be submitted and an error message will be displayed.

Note: The pattern attribute is not supported in all browsers, so it's important to test your forms in different browsers to ensure compatibility.

Also, it's important to use the pattern attribute in conjunction with other form validation techniques, such as server-side validation, to ensure that user input is properly validated and sanitized.

Overall, the pattern attribute is a useful feature for validating user input in forms, but it's important to use it correctly and ensure compatibility across different browsers.
